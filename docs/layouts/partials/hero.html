Ceres Icons

For the first time ever, Agrista has its own icon library, custom designed and built for our components and documentation. The icons are inspired by Streamline and Polaris.

Ceres Icons are designed to work with Ceres components, from form controls to navigation. Bootstrap Icons are SVGs, so they scale quickly and easily and can be styled with CSS. While they're built for Ceres, they'll work in any project.
